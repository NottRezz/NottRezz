## About Me

I'm a software development student who enjoys building things that actually do something. I've spent the past several years working on real projects, including backend systems, game mechanics, UI, and tools that people actively use.

I've done freelance work and built systems for clients, which taught me a lot about writing clean, maintainable code and thinking about performance and scalability early on. I've also run my own webstore for projects I've developed, so I've had experience with both the technical and user-facing side of things.
